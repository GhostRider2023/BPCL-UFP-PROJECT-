# assets/

## bpcl_logo.png

Drop the official BPCL logo here as `bpcl_logo.png` (or `.jpg` / `.svg`) and the
dashboard masthead picks it up automatically — no code change needed.

If no file is present, the masthead renders a neutral pipeline emblem instead.
We deliberately do **not** ship a facsimile of the BPCL corporate mark, so the
dashboard never misrepresents itself as officially branded artwork.

Recommended: ~200 px tall, transparent background.
